# AI Portfolio Website

A fully responsive, modern, and professional portfolio website built with React, Framer Motion, and Styled Components.

## Features

- **Responsive Design**: Looks great on all devices (Mobile, Tablet, Desktop).
- **Animations**:
  - Scroll-triggered reveals.
  - Hover effects on cards and buttons.
  - Parallax effects in Hero and Projects.
  - Typewriter effect in Hero.
- **Sections**: Hero, About, Skills, Projects, Education, Contact.
- **Theme**: Dark AI-themed aesthetics with glassmorphism.

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Navigate to the project directory:
   ```bash
   cd "New Prot"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

To start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173`.

### Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` directory.

## Customization

- **Personal Details**: Edit the text in `src/sections/*.jsx`.
- **Theme**: Update colors and fonts in `src/styles/theme.js`.
- **Global Styles**: Modify `src/styles/GlobalStyles.js`.
